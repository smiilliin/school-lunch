# School-Lunch - upload school lunch to instagram

This is a program that automatically uploads school meals to Instagram(data from "아이엠스쿨").

## Usage

Install packages

```bash
npm install
```

Input Instagram account and "아이엠스쿨" school ID

```env
ID=(Instagram ID)
PASSWORD=(Password)
SCHOOL_ID=(School ID)
```

Run

```bash
npm run start
```
